traduction en language morse

ce programme permet de traduire du texte en language morse, et inversément

pour ce faire, il suffit de rentrer du texte alphanumérique ou en language morse, 

pour ce dernier il faut noter les signaux courts par un point (.) et les signaux long par un tiret (-) un espace se verra remplacé par 3 espaces.
